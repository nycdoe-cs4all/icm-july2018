Unit 1. Learning Activity 4 . Repeat shapes, draw in grids

Objectives

This learning activity introduces the for loop, the most used control structure for repetition. First, the same shape is drawn several times by repeating a line of code, with a small variation in one parameter. This thought process is then replicated using a while loop (similar to the if statement students are already familiar with), and finally using a for loop. Two for loops are combined to create a row and a column, and nested to create a grid. Some creative possibilities of grids are explored: varying sizes, colors, and finally creating a grid made of the drawing students created in previous learning activities.

Suggested Duration

45 minutes

When a for loop is inside another for loop, we say they are nested.

Note that here we have added the variable x to avoid repeating i*30 four times (to draw the face, the left eye, the right eye, the mouth). If we did, the program would still work, but it would be harder to read and modify. The same goes for y, which stands for j*30. Using variables to avoid any repeating code is generally a good practice.
