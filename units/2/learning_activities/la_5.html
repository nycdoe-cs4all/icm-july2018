Unit 2. Learning Activity 5 . Translate, Rotate, Scale

Objectives

This learning activity introduces linear transformations: translation, rotation, and scale. Students also learn to compose them, creating an articulated arm, and to isolate them, using the push and pop functions.

Suggested Duration

40 minutes

Step by Step

An alternative to positioning elements by setting their coordinates directy is to change the position of the entire coordinate system: to move, rotate, and scale the x and y axis.

1. Translate

2. Rotate

Rotation happens in relation to the origin (0, 0), and takes an angle as a parameter. By default, this angle is in radians, but if using degrees is preferred, we can use angleMode(DEGREES).

4. Scale

Scaling increases or decreases the size of a shape. Increase by scaling by a number greater than 1, decrease by scaling by a number smaller than 1 (0.5 will decrease a shape's size by half)

5. Isolate transformations: push and pop

Transformations can be applied to a specific section of your code by using the push() and pop() functions. Push saves the current state of the coordinate system, and Pop restores it. Any calls to translate, rotate or scale will not affect any drawing done after the call to pop(). Notice how the red rectangle stays in the center of the screen, independently of where the mouse is:
